° LATCH SRH A VERROUILLAGE

Le latch SRH est une bascule qui mémorise 1 bit d'information.
- Pour H=1, cette bascule est dans son mode transparent et son fonctionnement
  est déterminé par ses entrées S et R.
- Pour H=0, cette bascule est dans son mode verrouillé.

Table de vérité
--------------|------------   
H    S    R   |   Q+  /Q+   
--------------|------------   
0    -    -   |   Q   /Q   mode mémoire
1    0    0   |   Q   /Q   mode mémoire
1    0    1   |   0    1   mise à 0
1    1    0   |   1    0   mise à 1
1    1    1   |   X    X   combinaison dite "interdite" à utiliser avec prudence
--------------|------------   
Pour cette bascule X=1. 
L'état initial de cette bascule est aléatoire.

Ne pas oublier que c'est la manière de sortir de cette combinaison qui doit être
choisie avec précaution (ne jamais passer directement de R=S=1 à R=S=0).

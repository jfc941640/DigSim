** Important information **

* When using appletviewer, make sure your CLASSPATH is set to the directory
  containing the DigSim *.class files
* start this applet with:
  appletviewer DigSim.html
* You can run this applet also with Netscape Navigator 2.0, only local
  loading and saving of schematics will be disabled.
* For the latest release of DigSim, go to my homepage:
  http://www.bart.nl/~ivr/digsim/index.html
